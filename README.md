## Operating Systems
- Windows
- Linux

## Installation
### Windows
- Download and install python from python.org
- Clone this repository on to your local device
- `pip install pywin32`

### Linux (ubuntu)
 $ `sudo apt update`
 
 $ `sudo apt install python git python-pip`
 
 $ `git clone https://github.com/e811-py/GlitchMic`
 
 $ `cd GlitchMic`
 
 $ `pip install pyalsaaudio`
 

## Usage
`python GlitchMic.py`
